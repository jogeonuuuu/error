RETR_EXTERNAL : 객체 바깥쪽 외곽선만 검색. 계층 구조는 만들지 않음.

RETR_LIST : 객체 바깥쪽과 안쪽 외곽선을 모두 검색. 계층 구조는 만들지 않음.

RETR_CCOMP : 모든 외곽선을 검색하고 2단계 계층 구조를 구성.

RETR_TREE : 모든 외곽선을 검색하고 전체 계층 구조를 구성.
